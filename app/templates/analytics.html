Peptide-HLA/MHC Binding Affinity Prediction

ⓘ The following downloadable CSV files contain majority-voted binding predictions. Each file includes all peptides from your uploaded data, along with predictions from all selected tools. In addition, the files contain tool-specific prediction columns such as ranks, binding levels, affinity scores, cores, and other outputs provided by each prediction tool. This ensures that, while you can easily see which peptides are majority-voted binders, you also retain full access to the individual predictions from each tool for every peptide. Binding prediction has been applied to a peptide list after removal of peptides labelled as #CONTAM or #DECOY, PTMs (for example, “A[+28.03]TFGVDESNAK” or “A(+28.03)TFGVDESNAK” are treated as “ATFGVDESNAK”), and resulting duplicates. For Class I, 8–14-mers were used as input, whereas for Class II 12–20-mers were used.

Key columns in these files include:

  • Peptide – the original peptide sequences you uploaded
  • StrippedPeptide – the peptide sequence after removal of PTM
  • Is Majority Voted Binder – indicates whether the peptide was identified as a majority-voted binder
  • Control – specifies whether the peptide is also present in the control group (if uploaded)

Violin plot

ⓘ The violin plot visualises the distribution of predicted binding values for peptides that are a subset of the binders shown in the UpSet plot. Only peptides within tool-specific thresholds for strong or weak binders are included. Thresholds vary depending on the prediction tool and MHC class, for example:

MHC Tool Peptide length Output column Strong binder threshold Weak binder threshold
Class I NetMHCpan-4.2 8–14-mers Rank 0.50% 2.00%
MixMHCpred 3.0 %Rank_bestAllele 2 10
MHCflurry 2.0 presentation_percentile 0.2% 5%
Class II NetMHCIIpan-4.3 12–20-mers %Rank_EL 1.00% 5.00%
MixMHC2pred-2.0 %Rank_best 2 10
  • X-axis: Sets (Sample Only) and overlaps (Sample1 ∩ Sample2), sorted from the largest to the smallest number of binders.
  • Y-axis: Predicted binding values for the peptides within the thresholds.
  • Violin shape: Shows the distribution density of binding values.
  • Box line: Indicates the median and interquartile range.
  • Points: Individual peptides are displayed as scatter points for detailed inspection.

This violin plot complements the UpSet plot by allowing for a comparison of binding value distributions between unique and shared peptide binders within tool-specific thresholds.
